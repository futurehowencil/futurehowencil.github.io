2.安装docker-mysql

  1. 拉取mysql镜像

  1. 创建并启动mysql容器

命令解释：

  • -i : 交互式运行容器
  • -t : 创建伪终端
  • -p 3306:3306 : 映射容器的3306端口到宿主机的3306端口，外部服务可以通过ip:3306访问mysql
  • –name : 容器的名称
  • -e MYSQL_ROOT_PASSWORD=123456 : 设置mysql root用户的密码
  • -d : 后台运行
